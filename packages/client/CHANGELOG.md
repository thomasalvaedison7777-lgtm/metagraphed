## [0.4.1](https://github.com/JSONbored/metagraphed/compare/client-v0.4.0...client-v0.4.1) (2026-06-16)

Republish of the 0.4.0 SDK (retries, ETag caching, convenience methods +
`fetchAll`). The `client-v0.4.0` release tag was created, but its npm publish was
blocked when `main` advanced to the platform 0.8.0 release commit before the
publish job ran, so the release-version guard refused the drifted commit. No code
changed between 0.4.0 and 0.4.1.

## 0.2.0

- Typed TypeScript client for the metagraphed API, generated from the OpenAPI
  contract. Adds the typed health-trends endpoint over the `0.1.0` bootstrap.
- Published to npm via OIDC trusted publishing (provenance, no long-lived token).
